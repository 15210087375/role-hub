## 质量控制标准

### 去AI味标准
- 口语化表达，避免书面语
- 短段落（每段不超过8行）
- 少修饰词，使用具体描述
- 不用破折号进行解释说明
- 增加人类思维特征

### 角色一致性标准
- 行为符合性格和动机
- 角色应随时间合理成长
- 关系应有来有往，避免单向
- 避免完美或纯恶角色
- 反派应有合理动机

### 世界真实性标准
- 动态世界，地点随时间变化
- 合理奇遇，非主角专属
- 经济系统合理流通
- 社会结构合理分布
- 时间流逝有真实感

## 使用流程

### 快速开始
```bash
# 创建新项目
cd tools
python project-init.py 我的小说

# 进入项目
cd ../projects/我的小说

# 检查一致性
python ../../tools/consistency-check.py
```

### 角色切换
- 架构师: `小说架构师模式`
- 主笔: `小说主笔`
- 责编: `小说责编`
