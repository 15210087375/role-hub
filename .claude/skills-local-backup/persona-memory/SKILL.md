# 行为记忆系统

## 系统概述
一个从日常AI交互中抽象提炼用户价值观、性格特征、心智模式、思维模式、工作习惯、行为偏好、决策框架、关注方向的分层记忆系统。

## 核心功能

### 1. 分层记忆沉淀
- **L0 状态层**: 原始对话记录
- **L1 情境层**: 对话场景和上下文
- **L2 行为层**: 可观察的行为模式
- **L3 认知层**: 思维方式和决策逻辑
- **L4 核心层**: 价值观和性格本质

### 2. 主动意图输入
- **目标与规划**: 用户设定的目标和计划
- **偏好与要求**: 明确的偏好和要求
- **约束与边界**: 限制条件和边界

### 3. 设备配置管理
- 设备信息和环境配置
- 文件路径和工具配置
- 工作流和自动化配置

### 4. 数据迁移同步
- 跨设备记忆库迁移
- 备份和恢复功能
- 版本控制和同步

## 技术实现

### 核心组件
1. **persona_core.py** - 核心引擎，实现分层沉淀和配置管理
2. **opencode_integration.py** - OpenCode集成接口，提供命令处理
3. **auto_runtime.py** - 会话生命周期钩子（会话开始加载上下文、结束写回记忆）
3. **配置管理** - YAML格式配置文件，支持动态更新
4. **数据存储** - JSON/YAML格式，本地存储，支持导出导入

### 分析能力
- **情境分析**: 识别对话类型、情感基调、环境信息
- **行为分析**: 提取提问模式、用词偏好、行为特征
- **认知分析**: 识别思维模式、决策框架、问题解决方式
- **核心分析**: 提炼价值观、性格特质、原则信念

## 隐私保护
- 所有数据存储在本地 `~/.behavior-memory/`
- 无云端同步（可选配置）
- 用户完全控制数据访问和删除
- 支持加密存储敏感信息

## 快速开始

### 1. 初始化系统
系统首次使用时自动初始化，创建所有必要的目录和配置文件。

### 自动模式（已启用）
- 已安装 `opencode.cmd` 包装器，自动调用会话前后记忆钩子
- 你正常使用 `opencode` 即可，不需要手动执行 start/end 命令
- 卸载包装器：`python ~/.behavior-memory/uninstall_opencode_wrapper.py`

### 每周复盘（已启用）
- 计划任务：`BehaviorMemoryWeeklyReview`（每周日 21:00）
- 自动输出：`~/.behavior-memory/Meta/复盘记录/<year-week>.yaml|md`
- 手动触发：`python ~/.behavior-memory/persona_core.py --weekly-review --days 7`

### 每月报告（已启用）
- 计划任务：`BehaviorMemoryMonthlyReport`（每月1日 21:15）
- 自动输出：`~/.behavior-memory/Meta/复盘记录/<year-month>.monthly.yaml|md`
- 手动触发：`python ~/.behavior-memory/persona_core.py --monthly-report --months 1`

### 异常检测与提醒（已启用）
- 自动触发：每次会话写入时检测路径偏离/风格偏离/可执行性偏离
- 队列文件：`~/.behavior-memory/Meta/洞察队列.yaml`
- 提醒文件：`~/.behavior-memory/Meta/异常提醒.md`
- 日常刷新任务：`BehaviorMemoryDailyAnomalyReminder`（每天 21:30）
- 手动刷新：`python ~/.behavior-memory/persona_core.py --refresh-anomaly-reminder`

### 迁移演练（已验证）
- 演练脚本：`python ~/.behavior-memory/migration_drill.py`
- 演练流程：导出 -> 新目录导入 -> 一致性校验
- 结果文件：`%TEMP%/persona_migration_drill_result.json`

### 0. 数据与技能分离（重要）
- 记忆数据目录：`~/.behavior-memory/`
- 技能执行目录：`~/.claude/skills/behavior-memory/`
- 技能命令读写的都是 `~/.behavior-memory/` 的数据

### 自动记忆钩子（可选）

## 故障排除

### 常见问题
1. **命令不生效**: 检查命令格式，使用 `/记忆系统 帮助` 查看正确用法
2. **配置更新失败**: 检查配置名和键路径是否正确
3. **文件权限问题**: 确保对 `~/.behavior-memory/` 目录有读写权限
4. **编码问题**: 系统使用UTF-8编码，确保终端支持

### 数据恢复
- 定期使用 `/记忆系统 备份` 创建备份
- 备份文件存储在 `~/.behavior-memory/Backups/`
- 支持导出完整记忆库
