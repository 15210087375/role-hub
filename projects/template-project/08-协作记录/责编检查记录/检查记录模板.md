## 专项检查结果

### 世界真实性检查
#### 时间流逝合理性
- [ ] 长时间跨度后，角色/地点变化合理
- [ ] 故地重游时，变化符合时间间隔
- [ ] 季节描写与时间匹配
- [ ] 无时间矛盾点

#### 奇遇合理性
- [ ] 奇遇进入条件具体合理
- [ ] 有前人尝试痕迹
- [ ] 进入有相应代价
- [ ] 非主角专属定制

#### 配角自主性
- [ ] 重要配角有独立动向
- [ ] 配角间有独立互动
- [ ] 配角会自主发展
- [ ] 无明显的工具人痕迹

#### 世界运转
- [ ] 有主角未参与的事件提及
- [ ] 世界在独立发展
- [ ] 无明显的"世界静止"问题
- [ ] 传闻系统使用合理

### 角色一致性检查
#### 性格与行为一致性
- [ ] 角色行为符合性格设定
- [ ] 反派/好人动机合理
- [ ] 角色转变有充分铺垫
- [ ] 关系变化符合逻辑

#### 身份与地点一致性
- [ ] 角色身份变化有合理解释
- [ ] 地点移动时间合理
- [ ] 无同时出现在两地的矛盾
- [ ] 能力使用在设定边界内

#### 人性复杂度平衡
- [ ] 复杂度在读者接受范围内
- [ ] 有明确的情感锚点角色
- [ ] 梯度设计合理
- [ ] 未触碰读者价值观底线

### 口语化质量检查
#### 读者定位适配
- [ ] 语言复杂度匹配目标读者
- [ ] 情绪通过动作/对话直接表现
- [ ] 冲突相对明确
- [ ] 目标简单清晰

#### 感官描写控制
- [ ] 动作优先于感官描写
- [ ] 细腻描写只在关键转折点
- [ ] 避免眼神、手指等细节堆砌
- [ ] 情绪外化为具体行为

#### 具体化程度
- [ ] 避免抽象虚词堆砌
- [ ] 虚词后接具体内容
- [ ] 使用具体名词和动词
- [ ] 描述具体不抽象

#### AI味检测
- [ ] 四字成语使用适度
- [ ] 排比句式不过度
- [ ] 逻辑连接词密度合理
- [ ] 句子结构不工整
- [ ] 适当留白不追求完整交代

### 毒点雷点检查
#### 内容毒点
- [ ] 无虐主无爽点
- [ ] 无绿帽情节
- [ ] 无憋屈金手指
- [ ] 无降智反派
- [ ] 无其他网文常见雷点

#### 价值观毒点
- [ ] 符合主流价值观
- [ ] 不宣扬错误价值观
- [ ] 不触碰敏感话题
- [ ] 不引起读者反感

### 节奏与爽点检查
#### 节奏控制
- [ ] 章节有明确进展
- [ ] 压抑与释放平衡
- [ ] 信息释放节奏合理
- [ ] 无拖沓或仓促

#### 爽点设计
- [ ] 本章有至少1个爽点
- [ ] 爽点有适当铺垫
- [ ] 爽点类型多样
- [ ] 爽点效果明显

#### 钩子设计
- [ ] 开篇有吸引力
- [ ] 结尾有下章钩子
- [ ] 钩子类型轮换
- [ ] 钩子自然不刻意
